# **App Name**: SeaPilotPro

## Core Features:

- Engine Log Input: Allows users to input engine log data, including date, time, and various engine parameters.
- Inventory Management: Enables tracking of spare parts and other inventory items, including quantity, location, and category.
- Performa Tracking: Tracks vessel performance data such as fuel consumption, speed, and voyage details.
- Data Synchronization: Allows users to import, export, and sync data with a server or local files for backup and sharing.
- Snapshot Saving: Allows users to take snapshop images of the performed calculations using html2canvas. Snapshots can be used to look up historical information of performed calculations.
- Anomaly Detection Tool: AI-powered tool that analyzes engine log data and identifies potential anomalies or deviations from normal operating parameters, suggesting possible causes and recommended actions. The tool will look for drops in pressure, spikes in temperature or pressure, changes in fuel usage etc.

## Style Guidelines:

- Primary color: Deep blue (#2E5CB8) to convey trust and stability, aligning with the maritime theme.
- Background color: Light gray (#F0F4F8) for a clean, modern look, ensuring readability and reducing eye strain in a dark theme.
- Accent color: Vivid orange (#FF7A00) to highlight important interactive elements, providing contrast and drawing attention to key actions.
- Body font: 'Inter' sans-serif for clear, modern text. Headline font: 'Space Grotesk' sans-serif for headers, titles and call-to-action text.
- Code font: 'Source Code Pro' monospaced, to display monospaced characters correctly.
- Consistent use of simple, line-based icons from Lucide to maintain a clean and modern aesthetic.
- Card-based layout for a modular and organized presentation of data, making information easily digestible.
